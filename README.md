# DARK-VIBE for Mailspring

DARK-VIBE is a modern UI theme for Mailspring built around deep dark surfaces, a vibrant blue accent, and subtle visual effects.

## Overview

This theme is designed for users who want a premium dark interface without losing readability.

- Dark layered backgrounds for better visual hierarchy
- Electric blue accent (`#4d9fff`) for active and interactive elements
- Soft glow effects on focus and important actions
- Subtle texture and shadow work for depth
- Refined typography and spacing for cleaner reading/composing

## Visual direction

DARK-VIBE combines:

- Charcoal/navy panels (`@background-primary`, `@background-secondary`)
- Bright blue highlights (`@accent-primary`)
- High-contrast but comfortable text colors (`@text-color`)
- Elevated cards, toolbar depth, and focus states for key interactions

## Installation

1. Clone this repository:

```bash
git clone https://github.com/Sud0ck3rs/DARK-VIBE.git
```

2. Open Mailspring and go to `Edit` -> `Install Theme...`
3. Select the downloaded `dark-vibe` folder
4. Restart Mailspring if the theme is not applied instantly

## Compatibility

- Mailspring: `*` (see `package.json`)
- Theme type: `ui`

## Project structure

- `styles/ui-variables.less`: color palette, typography, and core variables
- `styles/index.less`: component-level styling and visual effects
